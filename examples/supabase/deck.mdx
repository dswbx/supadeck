<Tag>What it is</Tag>
## Supabase *supplement*
<Divider />
<CardGrid columns={3}>
   <Card title="For AI builders">
      <p>
         Built for local and prototype workflows in tools like Lovable, Bolt,
         and similar AI-first products.
      </p>
   </Card>
   <Card title="~60% feature parity">
      <p>
         Enough of the common surface area to build and iterate quickly without
         chasing edge-case parity.
      </p>
   </Card>
   <Card title="Seamless upgrade">
      <p>
         When you are ready for production, moving to Supabase should require no
         code changes.
      </p>
   </Card>
</CardGrid>

## vs *Supabase*

<Divider />

| Metric |supalite | Supabase (t4g.nano) |
| ------ | -------- | ------------------- |
| Idle cost | **Storage only** – $0.20/gb <br /> <small className="dim">\~0.5mb per small project</small> | **~$0.71** compute before pausing <br /> <small className="dim">$0.0042/hr × 168h (7 days) · us-east-1</small> |
| Scale to zero | **Yes** – after 10s inactivity | Pauses after ~7 days |
| Cold start | **Yes** – **\<1s** | up to minutes |
| 200 idle projects | **\~$0.01** (storage only) <br /> <small className="dim">scales to zero after 10s</small> | **\~$142** (compute) <br /> <small className="dim">200 × $0.71 before each pauses</small> |
